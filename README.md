# tp_consecionaria
Proyecto en Java que simula el funcionamiento básico de una concesionaria de vehículos. Al ejecutarse, el programa carga una lista de autos y motos y muestra en consola información sobre cada uno, junto con distintos resultados procesados.
El proyecto hace uso del framework Collections y de la API Stream. También se construyó utilizando Spring Boot y se gestionó con Maven. Para reducir código repetitivo y favorecer buenas prácticas de desarrollo, se incorporó Lombok en la definición de las clases.
